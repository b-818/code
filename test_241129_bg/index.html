이미지의 크기 조절

이미지 태그로 코딩한 이미지는 너비나 높이 하나만 지정해주면 비율에 맞게 리사이징 됨 (이미지 크기 조절할 때 너비/높이 동시에 주면 안됨) 이미지의 크기를 px 또는 % 등으로 조절할 수 있는데, %는 이미지를 감싸는 요소의 너비를 기준으로 계산함 (원래 이미지 크기와 상관 없음)

  • 글자 글자

object-fit:cover;

메인페이지 최근 게시판에 이미지가 보이는 경우, 담당자가 비율에 맞지 않는 이미지를 주더라도 디자인한 영역에 이미지를 찌그러지지 않게 구현하는게 중요함 object-fit:cover;는 영역에 이미지를 위아래 가운데, 좌우로 가운데 정렬

  • 이미지 이미지

  • 글자 글자

object-fit:contain;

메인페이지 최근 게시판에 이미지가 보이는 경우, 담당자가 비율에 맞지 않는 이미지를 주더라도 디자인한 영역에 이미지를 찌그러지지 않게 구현하는게 중요함 object-fit:contain; 정해진 영역에 이미지를 잘리지 않게 넣을 때 사용

  • 이미지 이미지

  • 글자 글자

background-size:cover;

배경을 주는 요소에 빈 공간없이 배경 이미지를 꽉 채우는 방식 단, 이미지가 잘릴 수 있음

  • 글자 글자
  • 글자 글자

background-size:contain;

배경을 주는 요소에 배경이미지를 짤리지 않게 모두 표시하는 방식, 배경의 일부에는 배경이미지가 들어가지 않음(빈 곳이 발생)

background-size: 직접 입력 ;
