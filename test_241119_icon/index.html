png 이미지와 svg 이미지의 차이점

  • png 이미지

    확대하면 깨지는 이미지 형식, 모바일에서 선명하지 않음. 만약 png로 저장한 아이콘을 모바일에서 선명하게 보이게 하려면 2배 이상 사이즈로 저장해야함

  • svg 이미지

    아이콘이나 로고 같은 일러스트(벡터 기반)에서 제작한 이미지 저장가능(사진 불가) 모바일에서도 선명, 확대해도 깨지지 않음

svg를 이미지태그로 코딩한 스타일

아이콘을 이미지태그로 코딩하면 마우스 오버했을 때 이미지 교체해야하는 디자인이라면 구현 불가능하다. (진짜 안되는건 아닌데 복잡함)

폰트어썸은 유료화되어 사용 줄어듬
SI업체 솔루션 개발이나 어플리케이션 개발에서는 많이 활용(홈페이지는 현재 인기 없음)

  • 1.구글아이콘 홈페이지에서 아이콘 셋팅(두께나 스타일 지정) 아이콘 아무거나 클릭(오른쪽에 코드 복사)
  • 2.오른쪽 패널에서 static icon font 하단에 있는 css 링크 주소 복사해서 넣기(단 한번만 넣어주면 됨)
  • 3.실제 홈페이지에서 사용할 아이콘 찾기
  • 4.그 아이콘의 Inserting the icon 하단의 코드 복사
  • 5.그걸 아이콘이 들어갈 위치에 붙여넣음
  • 6.span태그는 i태그로 변경. 절대 class명 수정 x. 영문도 함부로 변경 안됨. >그걸 바꾸면 아이콘이 바뀜
  • 7.css에서 크기와 색상 변경 가능
